Requirements
============
Drush make: http://drupal.org/project/drush_make
For more information about make files read "Drush make theory for happy profile development" at
http://drupal.org/node/1006620

Installation
============
Copy To root of your drupal site (is_profile_2016.make)
drush make is_profile_2016.make

Copy is_profile_2016/modules folder to sites/all/


For image optimization follow this tutorial:
sudo apt-get install pngcrush jpegoptim
http://realityloop.com/blog/2015/08/12/automating-image-optimisation-drupal-improve-performance
